Tananã

Tananã é um software educativo para aprender a ler partituras. Como está numa fase inicial de desenvolvimento, só lê partituras simples com clave de sol, notas e silêncios. Ligaduras e triplets, por exemplo, não são suportadas ainda.

O formato de arquivo que Tananã lê é o MusicXML, um padrão usado mundialmente para partituras musicais. Aqui, por exemplo, podem ser encontrados programas que sabem ler/escrever o formato MusicXML.

Se você está em dúvida, abra o exemplo inicial para ver como funciona
